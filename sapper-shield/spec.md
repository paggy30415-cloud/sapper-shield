# Sapper-Shield V2: AI 時代的事實鑄幣廠與驗證電路規格書

## 1. 核心使命 (Mission)
本公司拒絕流於「替代人力」的文字去冗。在 AGI 時代，我們利用 Agent 的超人類算力，以 AI 產出的垃圾為原始礦石，透過大規模物理與邏輯熔煉，為傳統業主建立「外部錨點」。我們不競爭算力，我們競爭的是對真理結構的品味，成為信噪比的守門人。

## 2. 原生型工兵團隊與自動除錯工作流 (Native Workflow)
本系統的核心是由 Karpathy 的「驗證電路」與 Hassabis 的「深度搜索」交叉重組的原生架構。

### 2.1 Cleaner (物理事實溯源者 - Hassabis 深度轉型)
- **屬性**：原生型 Specialist（只有 AI 在場才可能實現的巨量搜索）
- **職責**：不再人工過濾文字，而是將客戶的雜亂資訊碎片，強制與外部物理世界的科學事實、原始單據、歷史日誌進行「百萬級併發量」的深度溯源與對齊。

### 2.2 Hardener (事實驗證電路 - Karpathy 機制轉型)
- **屬性**：原生型 Workflow / 邏輯驗證電路
- **職責**：為 Agent 建立一個可運行的外部強化學習（RL）獎勵環境。Agent 產出的每項營運數據，必須通過此「驗證電路」的電擊測試與自發性除錯，直到邏輯誤差小於 10^-6。

### 2.3 Auditor (品味與意義編譯器 - Reframer)
- **屬性**：人類真理判準的主導器
- **職責**：將「人類對真實性的執著、對嚴密性的品味」編譯成主導 Agent 執行的目標函數。推翻舊有框架，將混亂資訊重新編譯成高密度的知識金塊。

## 3. 改造前後 Deliverable 結構性差異 (V1 vs V2)
- **V1 替代型產出**：幫客戶修改、去冗後的乾淨文字報告（省時間，但可被隨時取代）。
- **V2 原生型產出**：經由驗證電路千錘百鍊產出的 **「數據誠信護照」** 與 **「高密度真理知識庫」**（贏得戰爭，提供沒有 Agent 就不存在的極致真實）。
